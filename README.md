# Hyprland dots

- To install needed packages under Debian: sudo apt install hyprland hyprpaper hyprlock waybar wofi wlogout dunst kitty nautilus firefox-esr pipewire wireplumber pavucontrol wl-clipboard playerctl

- Debian is old, so testing sources need to be enabled for this to work! (at least with 13.3.0)

- Autoinstall.sh is work in progress, feel free to test, but might have some slight problems.
